# Toy Difficulty Adjustment

이 예제는 target 기반 Proof of Work에서 한 단계 더 나아가, 블록 생성 속도에 따라 난이도를 조정하는 구조를 학습하기 위한 코드다.

## 왜 난이도 조정이 필요한가?

채굴자의 연산 능력은 계속 변할 수 있다.

- 채굴자가 많아지면 블록이 너무 빨리 생성됨
- 채굴자가 줄어들면 블록이 너무 느리게 생성됨

따라서 블록체인은 일정한 평균 블록 생성 시간을 유지하기 위해 난이도를 조정해야 한다.

## 핵심 아이디어

목표 블록 시간이 있다고 가정한다.

```text
target_block_time = 2초
```
